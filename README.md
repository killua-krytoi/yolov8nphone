### Waarshuwing
Deze code werkt alleen op Linux, om code te gebruiken op Windows/MacOS kies `no-gpiozero` branch.

Als code werkt niet, probeer om code te kopieren van `legacy` branch.

## Installatie



Clone de repository:

`git clone https://github.com/killua-krytoi/yolov8nphone`

`cd yolov8nphone`

Installeer de afhankelijkheden:

`pip install -r requirements.txt`

## Uitvoeren

Start het script met:

`python main.py`

## Aanbevolen

Gebruik bij voorkeur een virtuele omgeving:

`python -m venv venv`

`source venv/bin/activate` - macOS/Linux

`venv\Scripts\activate` - Windows


Installeer daarna opnieuw de dependencies en voer het script uit.
